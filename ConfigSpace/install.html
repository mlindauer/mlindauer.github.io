Installation¶

Requirements¶

ConfigSpace requires:

Numpy

Manually¶

To install ConfigSpace from command line type the following commands in your bash terminal:

https://github.com/automl/ParameterConfigSpace.git

cd ParameterConfigSpace/

python setup.py install
