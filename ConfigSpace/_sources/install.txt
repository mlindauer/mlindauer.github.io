Installation
============

.. role:: bash(code)
    :language: bash

Requirements
------------
ConfigSpace requires:


`Numpy <https://pypi.python.org/pypi/numpy>`_


Manually
------------

To install ConfigSpace from command line type the following commands in your bash terminal:

	:bash:`https://github.com/automl/ParameterConfigSpace.git`

	:bash:`cd ParameterConfigSpace/`

	:bash:`python setup.py install`
